Overview

NORD-FKB is a large-scale dataset for instance segmentation in aerial images, specifically designed for forest knowledge base applications. The dataset contains high-resolution aerial images with precise instance-level annotations, making it an ideal benchmark for developing and evaluating instance segmentation algorithms in forest environments.

Description

NORD-FKB is a comprehensive benchmark dataset for instance segmentation in aerial images. This large-scale and densely annotated dataset contains multiple object instances across various categories in high-resolution images. The distinctive characteristics of NORD-FKB include:

  • Large number of high-resolution images
  • Multiple important and commonly occurring categories
  • Large number of instances per category
  • Significant object scale variation
  • Real-world aerial conditions with varying object orientations
  • Precise instance-level annotations
